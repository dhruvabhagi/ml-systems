# Courses

## Stanford CS229: Machine Learning

Stanford's flagship graduate ML course covering the mathematical and algorithmic 
foundations of modern machine learning. Topics include:
- **Supervised Learning** — linear and logistic regression, generative/discriminative 
  modeling, naive Bayes, SVMs, kernel methods, neural networks, backpropagation
- **Tree Methods & Ensembles** — decision trees, ensemble methods, boosting
- **Unsupervised Learning** — k-means, mixture of Gaussians, EM algorithm, factor analysis, ICA
- **Dimensionality Reduction** — PCA, kernel PCA, autoencoders
- **Regularization & Model Selection** — bias/variance tradeoffs, cross-validation, feature selection
- **Learning Theory** — VC theory, large margins, PAC learning
- **Probabilistic Modeling** — GLMs, exponential family, Gaussian processes
- **Optimization** — convex optimization, Newton's method, gradient descent variants
- **Reinforcement Learning** — MDPs, Bellman equations, value/policy iteration, 
  Q-learning, policy search, LQR/LQG

## CS-231N · Visual Computing & Deep Learning

Stanford's flagship graduate course on deep learning applied to visual recognition, 
covering both theory and systems-level implementation. Topics include:
- **CNN Fundamentals** — convolutional layers, pooling, receptive fields, filter 
  parameter analysis, transposed convolution
- **Network Architectures** — ResNet, U-Net, skip connections, RoI pooling, 
  anchor boxes, region proposal networks
- **Training & Optimization** — SGD, momentum, Adam, AdamW, learning rate 
  scheduling, cosine annealing, He initialization
- **Normalization & Regularization** — batch normalization, layer normalization, 
  group normalization, spatial batch normalization, dropout
- **Transformers & Attention** — self-attention, multi-head attention, positional 
  encoding, encoder-decoder dynamics, KV cache, causal masking
- **Vision Transformers** — ViT, patch embeddings, Swin Transformer, SegFormer
- **Sequence Models** — RNNs, LSTMs, teacher forcing, BPTT, image captioning
- **Generative Models** — autoencoders, VAEs, GANs
- **Object Detection & Segmentation** — semantic segmentation, instance segmentation, 
  pose estimation
- **Model Interpretability** — Grad-CAM, saliency maps, feature visualization
- **Video Understanding** — optical flow, TimeSformer, Video Swin, ViViT
